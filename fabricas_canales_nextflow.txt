Fábricas Básicas
Channel.of()
Utilidad: Crea canales desde valores específicos
Diferencia: Emite cada argumento como elemento individual
Channel.of(1, 2, 3, 'hello')  // Emite: 1, 2, 3, hello

Channel.fromList()
Utilidad: Crea canales desde listas
Diferencia: Similar a of() pero acepta explícitamente una lista
Channel.fromList(['a', 'b', 'c'])  // Emite: a, b, c

Channel.value()
Utilidad: Crea canales de valor único reutilizable
Diferencia: El valor se puede usar múltiples veces sin consumirse
Channel.value('reference.fa')  // Valor constante reutilizable

Fábricas para Archivos
Channel.fromPath()
Utilidad: Busca archivos usando patrones glob
Diferencia: Maneja archivos individuales y patrones
Channel.fromPath('*.fastq.gz')  // Todos los archivos FASTQ
Channel.fromPath('data/**/*.txt')  // Recursivo en subdirectorios

Channel.fromFilePairs()
Utilidad: Agrupa archivos en pares (ej: paired-end reads)
Diferencia: Emite tuplas [clave_agrupación, [archivo1, archivo2]]
Channel.fromFilePairs('*_{1,2}.fastq')
// Emite: [sample1, [sample1_1.fastq, sample1_2.fastq]]

Fábricas Especializadas
Channel.fromSRA()
Utilidad: Descarga datos directamente desde NCBI SRA
Diferencia: Conecta automáticamente con bases de datos públicas
Channel.fromSRA('SRP043510')  // Proyecto SRA completo
Channel.fromSRA(['SRR123', 'SRR124'])  // Accesiones específicas

Channel.empty()
Utilidad: Crea canal vacío (no emite valores)
Diferencia: Útil para condicionales y placeholders
Channel.empty()  // Canal sin datos

Fábricas Dinámicas
Channel.watchPath()
Utilidad: Monitorea directorios en tiempo real
Diferencia: Emite archivos cuando aparecen/cambian
Channel.watchPath('/data/*.txt', 'create,modify')

Channel.interval()
Utilidad: Emite valores a intervalos regulares
Diferencia: Para tareas periódicas o temporizadas
Channel.interval('5s')  // Emite 0, 1, 2... cada 5 segundos

Fábricas Avanzadas
Channel.topic()
Utilidad: Canales compartidos entre múltiples procesos
Diferencia: Permite comunicación indirecta entre procesos
Channel.topic('shared_results')  // Canal compartido

Channel.fromLineage() (Experimental)
Utilidad: Accede a archivos desde el sistema de linaje
Diferencia: Para workflows con trazabilidad de datos
Channel.fromLineage(workflowRun: 'lid://...', label: ['alpha'])

Fábricas Deprecadas
Channel.from() (Deprecado)
Reemplazo: Usar Channel.of() o Channel.fromList()
Razón: Sintaxis más clara y consistente
Resumen de Diferencias Clave
Factory	Tipo de Datos	Reutilizable	Caso de Uso Principal
of()	Valores individuales	No	Listas simples de valores
fromPath()	Archivos	No	Buscar archivos con patrones
fromFilePairs()	Pares de archivos	No	Datos paired-end
value()	Valor único	Sí	Referencias constantes
fromSRA()	Datos remotos	No	Datos públicos de secuenciación
watchPath()	Archivos dinámicos	No	Monitoreo en tiempo real
La elección de la factoría correcta depende del tipo de datos que manejes y si necesitas reutilización del canal.
